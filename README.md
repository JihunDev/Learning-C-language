# C 언어 완전 정복: 초급부터 고급까지

C 언어는 50년이 넘는 역사를 가진 언어이지만, 운영체제·임베디드·시스템 프로그래밍 분야에서 여전히 가장 중요한 언어입니다. 이 책은 **프로그래밍 경험이 전혀 없는 초보자**부터 **시스템 수준의 깊은 이해를 원하는 개발자**까지 모두를 대상으로 집필되었습니다.

## 이 책의 구성

| 부 | 내용 | 대상 |
|---|---|---|
| 제1부 | 프로그래밍 기초와 C 언어 입문 | 완전 초보자 |
| 제2부 | 기본 문법과 데이터 처리 | 입문자 |
| 제3부 | 제어 구조와 프로그램 흐름 | 입문자 |
| 제4부 | 함수와 모듈화 | 초급 |
| 제5부 | 메모리와 포인터 | 초·중급 |
| 제6부 | 배열과 문자열 | 초·중급 |
| 제7부 | 구조체와 사용자 정의 타입 | 중급 |
| 제8부 | 파일 처리와 실전 응용 | 중급 |
| 제9부 | 디버깅과 최적화 | 중·고급 |

## 이 책의 특징

- **단계적 학습**: 쉬운 개념부터 시작해 점진적으로 심화
- **실습 중심**: 각 장마다 실제 프로그램을 만드는 실습 포함
- **실전 예제**: 현업에서 바로 활용할 수 있는 코드 패턴 제공
- **C99/C11/C17 표준** 기준으로 작성
