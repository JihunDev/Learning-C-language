# A. ASCII 코드표

## A. ASCII 코드표

ASCII(American Standard Code for Information Interchange)는 7비트 문자 인코딩 표준으로, 0\~127 범위의 총 128개 문자를 정의합니다. C 언어에서 `char` 타입은 이 ASCII 값을 직접 저장하며, 문자 리터럴(`'A'`, `'\n'` 등)은 내부적으로 해당 정수값으로 처리됩니다.

***

### 제어 문자 (0–31, 127)

출력되지 않는 비가시 문자(non-printable character)입니다. C 언어에서는 이스케이프 시퀀스로 표현합니다.

| Dec | Hex | Oct |  기호 |  이스케이프 | 설명                                            |
| :-: | :-: | :-: | :-: | :----: | --------------------------------------------- |
|  0  |  00 | 000 | NUL |  `\0`  | 널 문자 (Null) — 문자열 종료 마커                       |
|  1  |  01 | 001 | SOH |        | 헤딩 시작 (Start of Heading)                      |
|  2  |  02 | 002 | STX |        | 텍스트 시작 (Start of Text)                        |
|  3  |  03 | 003 | ETX |        | 텍스트 종료 (End of Text)                          |
|  4  |  04 | 004 | EOT |        | 전송 종료 (End of Transmission)                   |
|  5  |  05 | 005 | ENQ |        | 조회 (Enquiry)                                  |
|  6  |  06 | 006 | ACK |        | 긍정 응답 (Acknowledge)                           |
|  7  |  07 | 007 | BEL |  `\a`  | 경보음 (Bell)                                    |
|  8  |  08 | 010 |  BS |  `\b`  | 백스페이스 (Backspace)                             |
|  9  |  09 | 011 |  HT |  `\t`  | 수평 탭 (Horizontal Tab)                         |
|  10 |  0A | 012 |  LF |  `\n`  | 줄 바꿈 (Line Feed) — Unix 개행                    |
|  11 |  0B | 013 |  VT |  `\v`  | 수직 탭 (Vertical Tab)                           |
|  12 |  0C | 014 |  FF |  `\f`  | 폼 피드 (Form Feed)                              |
|  13 |  0D | 015 |  CR |  `\r`  | 캐리지 리턴 (Carriage Return) — Windows 개행(`\r\n`) |
|  14 |  0E | 016 |  SO |        | Shift Out                                     |
|  15 |  0F | 017 |  SI |        | Shift In                                      |
|  16 |  10 | 020 | DLE |        | 데이터 링크 이스케이프                                  |
|  17 |  11 | 021 | DC1 |        | 장치 제어 1 (XON)                                 |
|  18 |  12 | 022 | DC2 |        | 장치 제어 2                                       |
|  19 |  13 | 023 | DC3 |        | 장치 제어 3 (XOFF)                                |
|  20 |  14 | 024 | DC4 |        | 장치 제어 4                                       |
|  21 |  15 | 025 | NAK |        | 부정 응답 (Negative Acknowledge)                  |
|  22 |  16 | 026 | SYN |        | 동기화 유휴 (Synchronous Idle)                     |
|  23 |  17 | 027 | ETB |        | 전송 블록 종료                                      |
|  24 |  18 | 030 | CAN |        | 취소 (Cancel)                                   |
|  25 |  19 | 031 |  EM |        | 매체 종료 (End of Medium)                         |
|  26 |  1A | 032 | SUB |        | 대체 (Substitute)                               |
|  27 |  1B | 033 | ESC | `\e` ¹ | 이스케이프 (Escape)                                |
|  28 |  1C | 034 |  FS |        | 파일 구분자 (File Separator)                       |
|  29 |  1D | 035 |  GS |        | 그룹 구분자 (Group Separator)                      |
|  30 |  1E | 036 |  RS |        | 레코드 구분자 (Record Separator)                    |
|  31 |  1F | 037 |  US |        | 단위 구분자 (Unit Separator)                       |
| 127 |  7F | 177 | DEL |        | 삭제 (Delete)                                   |

> ¹ `\e`는 GCC 확장입니다. 표준 C에서는 `'\x1B'` 또는 `'\033'`으로 표기합니다.

### C 언어 활용 팁

#### 문자 판별 (`<ctype.h>`)

```c
#include <ctype.h>

isdigit(ch)   // '0'~'9'  (48~57)
isalpha(ch)   // 'A'~'Z', 'a'~'z'
isupper(ch)   // 'A'~'Z'  (65~90)
islower(ch)   // 'a'~'z'  (97~122)
isspace(ch)   // 공백, '\t', '\n', '\r', '\v', '\f'
isprint(ch)   // 출력 가능 문자 (32~126)
iscntrl(ch)   // 제어 문자 (0~31, 127)
```

#### 대소문자 변환

```c
// 소문자 → 대문자: 비트 5를 끔 (0x20 = 32)
char upper = ch & ~0x20;    /* 비트 조작 방식 */
char upper2 = toupper(ch);  /* <ctype.h> 권장 */

// 대문자 → 소문자: 비트 5를 켬
char lower = ch | 0x20;     /* 비트 조작 방식 */
char lower2 = tolower(ch);  /* <ctype.h> 권장 */
```

#### 문자 ↔ 정수 변환

```c
/* 문자 숫자 → 정수 */
int n = '7' - '0';    /* n = 7 */

/* 정수 → 문자 숫자 */
char ch = 3 + '0';    /* ch = '3' */

/* 16진수 문자 → 정수 */
int hex_digit(char c) {
    if (c >= '0' && c <= '9') return c - '0';
    if (c >= 'A' && c <= 'F') return c - 'A' + 10;
    if (c >= 'a' && c <= 'f') return c - 'a' + 10;
    return -1; /* 유효하지 않은 입력 */
}
```

#### ASCII 값 출력 예제

```c
#include <stdio.h>

int main(void) {
    int i;
    printf("%-5s %-5s %-5s %s\n", "Dec", "Hex", "Oct", "Char");
    printf("--------------------------------\n");
    for (i = 32; i <= 126; i++) {
        printf("%-5d %-5X %-5o %c\n", i, i, i, (char)i);
    }
    return 0;
}
```
